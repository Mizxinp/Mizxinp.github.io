快速上手

1.引入mongoose模块

2.定义一个Schema：一种以文件形式存储的数据库模型骨架

3.连接数据库,指定连接的数据库不需要存在，当你插入第一条数据之后就会自动被创建出来(myserver是自己创建的)

4.设计文档结构，官方文档的

这里我用这样的来做简单例子

5.将Shcema发布为model

  • mongoose.model 方法就是用来将Shcema发布为 model
  • 第一个参数：传入一个大写名词单数字符串用来表示你的数据库名称，mongoose 会自动将大写名词的字符串生成 小写复数 的集合名称例如这里的 User 最终会变为 users 集合名称
  • 第二个参数：架构 Schema
  • 返回值：模型构造函数

当有了模型构造函数之后，就可以使用这个构造函数对users集合中的数据进行增删改查了
